# JSON 升级文件生成工具
Generate JSON update config file for app upgrade

## 功能 / Features
- 生成 APP 升级用 JSON 配置文件
- Generate JSON config for app update
- 手动推送到 GitHub 仓库
- Manually push to GitHub repository
- 支持 APP 版本检查与更新
- Support app version check and upgrade

## 使用 / Usage
运行程序，填写信息，生成 JSON 文件后手动上传到仓库。
Run the program, fill in information, generate JSON file and upload manually to repository.
